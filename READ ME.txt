Steps to run the project:

1) Install Xampp
2) Go to your xampp folder, open htdocs folder
3) Create a folder named "step-x"
4) Extract the contents of the zip inside this folder.
5) Start Xampp control panel. Start Apache and MySQL.
6) Go to your browser and open this link : http://localhost/step-x/admin/create_databse.php
7) Your databse and the necessary phpMyAdmin should be created automatically on your system
8) Now open this link : http://localhost/step-x/
9) Try and test it as you would be using a new website. Register your account, login, add to cart, checkout, do whatever you like.
10) if you login as admin, you would be redirected to the admin panel automatically, here you can see the orders and customer info.
